# 贝贝家纺小程序视觉优化设计

## 目标

在不改变第一版功能范围和页面结构的前提下，把小程序视觉从传统红棕促销风，调整为更年轻、清爽、亲和的生活方式家居风。

## 视觉方向

采用 `奶油白 + 鼠尾草绿 + 暖珊瑚`：

- 奶油白作为页面背景和卡片基底，减少旧版米黄和厚重阴影。
- 鼠尾草绿作为主品牌色，表达清爽、居家、年轻。
- 暖珊瑚作为行动按钮和价格强调色，保留转化力度。

## 调整范围

- 全局背景、文字、卡片和链接色。
- 首页和门店页首屏 hero。
- 商品卡片封面色块、价格、标签。
- 快捷按钮和底部联系栏按钮。
- 产品分类页 tab。
- 商品详情页标签、封面和价格。
- 门店页信息卡、门店实拍占位块和 FAQ。

## 设计约束

- 不改业务逻辑。
- 不新增线上支付、会员、购物车等功能。
- 不依赖真实图片；当前仍使用柔和渐变色块作为图片占位。
- 保持微信小程序原生 WXSS 可用。
